Welcome to my (super ghetto) API!

This API hosts IRS data for every state and zip code in the US. For basic queries, search using these url extensions:

Itemtag
All Data by Year/taxdata/(year)
Data by State & Year/taxdata/(year)/state/(state)
Data by Zip Code & Year/taxdata/(year)/zipcode/(zip)
Info on retrieving graphs & visuals/visuals/info
